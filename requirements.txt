Follow these steps to install dependencies and run the Telegram bot.

1. Create a virtual environment
python -m venv venv

2. Activate the virtual environment

Windows: venv\Scripts\activate

Linux or macOS: source venv/bin/activate

3. Install required packages
pip install -r requirements.txt

4. Create the config.py file

Create a file named config.py in the project root:

BOT_TOKEN = "your-telegram-bot-token"
ADMIN_IDS = [123456789]  # List of admin user IDs
MAX_CAPTION_LENGTH = 1024


Important:
Do not commit this file to GitHub. Make sure config.py is listed inside .gitignore.

5. Run the bot
python bot.py

6. Database initialization

The bot uses SQLite. The database file bot_data.db is created automatically on first run and includes automatic migrations when needed.
